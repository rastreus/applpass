# Examples

Shell script examples:

- `quick-start.sh`: add/get/list/delete workflow with temporary credentials
- `update-with-generate.sh`: update an existing secret using generated password
- `list-json.sh`: list personal/shared entries in machine-readable form

Run any script with an optional binary argument:

```bash
./examples/quick-start.sh .build/release/applpass
```

Agent skill example:

- `skills/applpass-commands/SKILL.md`: reusable command patterns for automated agents
