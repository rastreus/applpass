# Changelog

All notable changes to `applpass` are documented in this file.

## [Unreleased]

### Added
- GitHub Actions release workflow for tagged builds.
- Release artifact packaging (`.tar.gz` and SHA-256 checksum).
- Optional pre-commit formatting hook support.

## [0.1.0] - 2026-02-14

### Added
- Initial public release of `applpass` for macOS keychain password management.
- Command set: `get`, `list`, `add`, `update`, `delete`, and `generate`.
- Structured output formats: `table`, `json`, `csv`, and `plain`.
- Swift Testing unit/integration coverage and CLI smoke tests.
- User documentation, examples, and contributor workflow guides.
